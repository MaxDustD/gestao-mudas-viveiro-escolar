# ViveiroEscolar.Web

Run the Blazor Server app with hot reload using dotnet watch.

Start (recommended):

```bash
dotnet watch run --project src/ViveiroEscolar.Web/ViveiroEscolar.Web.csproj --urls http://localhost:5000
```

Or rely on `launchSettings.json` defaults and run:

```bash
dotnet watch run --project src/ViveiroEscolar.Web/ViveiroEscolar.Web.csproj
```

Open http://localhost:5000 in your browser.
